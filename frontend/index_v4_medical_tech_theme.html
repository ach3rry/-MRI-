DICOM 序列处理挑战

不同厂商设备输出格式差异大，多序列合并复杂， 缺失帧处理需要鲁棒性设计。

自动序列检测 + 统一 JSON 中间格式

MLLM 输出稳定性控制

大模型输出格式不稳定，JSON 结构容易出错， 标签一致性难以保证。

温度 0.01 + JSON Schema 强制校验 + 正则提取
