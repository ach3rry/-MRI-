Quick Start

使用指南

快速上手，几分钟内完成数据处理

1
环境准备
确保已安装 Python 3.8+，然后安装依赖：
pip install -r requirements.txt
下载并放置模型文件到 ./model 和 ./qwen3-1.7B 目录。
2
配置设置
在 config.py 中配置输入输出目录：
input_dir = './202505-202506'
test_set_dir = './测试集'
output_dir = './output'
3
运行流水线
运行完整流水线：
python main.py
或运行指定阶段：
python main.py --start-stage "DICOM转NIfTI" --end-stage "NIfTI转MP4视频"
4
查看结果
处理完成后，结果将保存在输出目录中：
  • CSV 文件：提取的结构化数据
  • JSON 文件：清洗和标注后的数据
  • MP4 视频：可视化的医学影像
  • 日志文件：详细的处理记录
5
高级选项
查看所有可用阶段：
python main.py --list-stages
跳过特定阶段：
python main.py --skip-stages "标签生成" "视频转换"
仅运行指定阶段：
python main.py --only-stages "医疗报告提取" "JSON数据清洗"
